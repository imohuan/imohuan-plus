## 命令

- 监听编译 `pnpm dev`
- `Esbuild`监听编译 `pnpm esbuild:dev`
- 项目编译 `pnpm build`
- 编译预览 `pnpm preview`
- 执行测试 `pnpm test`
- 发布 `npm login && npm publish`
